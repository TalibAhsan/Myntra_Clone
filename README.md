# Myntra Functional Clone

A front-end e-commerce clone inspired by Myntra. It displays a product catalogue and provides a functional shopping bag backed by browser local storage.

## Features

- Responsive product catalogue with pricing and ratings
- Add products to the shopping bag
- Bag item count in the header
- Persistent bag contents using `localStorage`
- Order summary with item removal support

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## Run Locally

1. Clone the repository.
2. Open `index.html` in a modern web browser.

No build step or package installation is required.

## Project Structure

```
├── css/          # Page styling
├── data/         # Product catalogue data
├── images/       # Product and brand images
├── pages/        # Shopping bag page
├── scripts/      # Site behaviour
└── index.html    # Home page
```
